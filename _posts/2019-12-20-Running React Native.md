# Running React Native

신규 앱 개발에 리액트 네이티브를 적용하고자하여 스터디를 시작합니다.

## Expo CLI Quickstart
Expo는 다양한 리엑트 툴을 미리 포함하고있어 처음 리액트 앱을 개발할 때 좋습니다. 
> Node 10 LTS 이상의 node.js가 설치되어있어야합니다.  

~~~
npm install -g expo-cli
~~~

초기 템플릿은 blank로 비어있는 상태를 선택합니다.
